# GBFRSiegfriedButBetter

nexus link

### Summary
The goal of this mod is to buff Siegfried's most underwhelming aspects while keeping the overall gameplay mostly the same. His worst attacks gain some new use cases, and his abhorrent SBA meter gain can be promoted to "below average" if he keeps landing those Perfect Executions.

### Major Changes:
* Link Attack allows Perfect Execution to be used afterward, regardless of combo state
* Draconic Release uncaps the damage of all attacks, but can only be used once per mission
* Verdrangen can start/restart combo chains like Uwe; time attack input after 2nd explosion
* L'Ombre d'Hier allows a timed attack to trigger Combo Finisher into Perfect Execution after a successful parry
* More SBA meter gain from landing Perfect Execution

### Minor Changes:
* Slightly more SBA meter gain from Uwe, Nelah Nav, and Lunge 
* Lunge, Aerial Barrage, and timed aerial attacks do more damage 
* Aerial Barrage has significantly higher Stun values
* Salvator cooldown decreased (110s->90s)

### Sigils:
* Dragonslayer's Dominance duration increased (10s -> 12s)
* Dragonslayer's Warpath defense reduction decreased (50% -> 40%)

### Random Shit:
Roll of the Die weightings improved

```1/5/18/26/50 -> 5/10/20/40/25 % chance``` to deal `4x/3x/2x/1x/0x` damage

Average damage multiplier goes from `0.81x` to `1.3x` to be more closely aligned with other damage-boosting sigils. ARE YA FEELIN LUCKY?
